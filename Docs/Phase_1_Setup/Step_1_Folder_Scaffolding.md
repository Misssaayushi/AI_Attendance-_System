# Step 1: AI Module Folder Scaffolding

## Objective
Establish the directory structure required for the modular AI system to ensure clean separation between logic, datasets, and logs.

## Tasks
- Create the main `ai_module/` directory.
- Create subdirectories:
  - `dataset/`: For storing raw student images.
  - `encodings/`: For storing serialized face data.
  - `logs/`: For runtime diagnostics.
  - `tests/`: For validation scripts.

## Expected Outcome
A clean filesystem hierarchy ready for Python module development.

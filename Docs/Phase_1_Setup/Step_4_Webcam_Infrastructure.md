# Step 4: Webcam Hardware Interface

## Objective
Build a robust, class-based interface for camera interaction to ensure stable frame capture and proper resource management.

## Tasks
- Add `CameraHandler` class to `ai_module/utils.py`.
- Implement `get_frame()`, `show_frame()`, and `cleanup()` methods.
- Add error handling for "Camera Not Found" or "Permission Denied" scenarios.

## Expected Outcome
A production-ready webcam system that can be easily "plugged in" to registration or recognition logic.
